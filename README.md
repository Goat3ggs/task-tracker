# Task Tracker

Task Tracker is an application that lets you create, organize, and follow your tasks on a daily basis.

## Demo

[Live Demo](https://goat3ggs.github.io/task-tracker/)

##Screenshot

![Task Tracker Screenshot](demo-screenshot.png)

## Features

- Add task
- Delete task
- Edit task
- Mark as completed
- Filter tasks by status
- Search tasks by title
- save tasks to localStorage

## Tech Stack

- HTML, CSS, JavaScript, ES Modules, localStorage
